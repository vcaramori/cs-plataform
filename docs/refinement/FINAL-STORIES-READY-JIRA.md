**Description:**  
Daily cron job analyzes customer interactions and NPS responses to extract sentiment, themes, and quotes. Uses Gemini for NLP analysis.

3. ✅ Fetches interactions + NPS from last 24h WHERE `sentiment_score IS NULL`

6. ✅ Updates `interactions.sentiment_score` with value
7. ✅ Inserts into `interaction_themes` table (interaction_id, theme, frequency)
8. ✅ Deduplication: in-memory Set of processed interaction IDs to prevent double-processing
9. ✅ Logging: if > 1000 interactions processed, log warning (possible double run)
10. ✅ Error handling: if Gemini rate limit hit, implement exponential backoff (1s → 2s → 4s)
11. ✅ Timeout: maxDuration=300s (5 minutes)
12. ✅ Logging: start time, count, end time in logs
