# Manual do Usuário - Detalhe do Cliente (Account Detail)

Bem-vindo ao manual da página de **Detalhe do Cliente** do CS-Continuum. Esta tela oferece uma visão 360º de um cliente específico, consolidando dados financeiros, saúde, adoção e interações.

---

## 🎯 Objetivo da Tela
Permitir que o CSM ou Gestor entenda profundamente o momento do cliente, identifique riscos de churn, oportunidades de expansão e histórico de relacionamento.

---

## 🔝 Cabeçalho (Header) e Indicadores Rápidos

O topo da página traz as informações mais críticas do cliente de forma resumida:

### 1. Identificação do Cliente
- **Nome**, **Segmento** (Indústria, MRO, Varejo) e **ID**.
- **Ícone de Lápis**: Permite editar os dados básicos do cliente.

### 2. Receita Mensal (Líquida)
- **O que é**: O MRR real que o cliente paga hoje.
- **Diferencial**: Ao passar o mouse sobre o valor, você verá o detalhamento (Valor Nominal vs Desconto Ativo). Este valor é calculado dinamicamente com base nas regras de governança comercial.

### 3. Cronômetro de Renovação
- **O que é**: Quantos dias faltam para o contrato expirar ou precisar ser renovado.
- **Cores**: Muda de verde para vermelho conforme a data se aproxima. Se faltarem menos de 90 dias, um botão "Preparar Renovação" aparecerá para te ajudar a iniciar o processo.

### 4. Índice de Saúde (Health Score)
- **O que é**: A nota atual de saúde do cliente (0 a 100).
- **Gráfico de Fundo**: Mostra a tendência das últimas semanas.
- **Ações**:
  - Clique no card para abrir o histórico detalhado.
  - Clique no ícone de faísca (✨) para **Gatilhar Análise Cognitiva**. A IA lerá todas as interações recentes e sugerirá uma nota (Shadow Score).

### 5. Mini-Gauges (Métricas de Apoio)
- **Adoção**: Percentual de uso das funcionalidades contratadas.
- **Chamados**: Saúde do suporte (tickets abertos vs resolvidos).
- **Relacionamento**: Sentimento das últimas reuniões e e-mails.
- **NPS**: Nota de satisfação dada pelo cliente.
- **SLA**: Se os níveis de serviço estão sendo cumpridos.
