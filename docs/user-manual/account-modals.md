# Manual do Usuário - Modais de Apoio (Accounts)

Este manual detalha o funcionamento dos modais que você pode abrir a partir da tela de Detalhe do Cliente. Eles servem para gerenciar informações profundas de adoção e histórico.

---

## 🛠️ Modal: Adoção Funcional

Este modal permite que você mapeie exatamente quais partes da plataforma o cliente está usando e onde ele está travado.

### Como Usar:
1. Clique no ícone de engrenagem ou "Adoção" no cabeçalho do cliente.
2. Na barra lateral esquerda, você verá a lista de todas as funcionalidades/módulos contratados.
3. Clique em uma funcionalidade para abrir o formulário de edição na direita.

### Campos e Regras de Negócio:
- **Status de Adoção**:
  - `Não Iniciado`: O cliente ainda não começou a usar.
  - `Uso Parcial`: Usa algumas funções, mas não todas.
  - `Em Uso`: Sucesso! Cliente usa plenamente.
  - `Bloqueado`: O cliente quer usar, mas há um impedimento.
  - `Não Aplicável`: Esta função não faz sentido para a operação deste cliente.
- **Responsável (CSM)**: Quem do nosso time está acompanhando essa adoção.
- **Data Alvo**: Prazo para que o cliente atinja o status "Em Uso".

### ⚠️ Regra Especial para Bloqueios:
Se você marcar o status como **Bloqueado** ou **Não Aplicável**, o sistema abrirá um campo obrigatório de **Justificativa** e **Plano de Ação**. Você deve descrever o que está travando o cliente (Ex: Integração pendente) e quais passos vamos tomar para resolver.

---

## 📝 Modal: Detalhes da Interação (Registro de Atividades)

Este modal serve para você revisar ou editar o histórico de conversas, reuniões e e-mails com o cliente.

### Recursos:
- **Estratégia e Relacionamento**: Mostra o tipo de interação (Reunião, E-mail, QBR, etc.).
- **Apontamentos**: O texto completo do que foi conversado ou a ata da reunião.
- **Edição**: Se você precisar corrigir alguma informação, clique em "Editar Log" no topo do modal.

### Dica de Uso:
Se a interação foi gerada automaticamente por uma transcrição de IA, você verá uma área no rodapé chamada "Contexto Operacional" e "Insight Original". Use essas informações para entender o resumo da conversa sem precisar ler todo o texto.
