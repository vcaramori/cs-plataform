# Manual do Usuário — Suporte (Avaliação 2.0)

Bem-vindo ao manual da tela de **Suporte (Help Desk / Chamados)**. Esta é a central de atendimento para resolver dúvidas e problemas dos clientes.

---

## 🎯 O que esta tela faz?

Esta tela lista todos os chamados de suporte abertos pelos clientes. Você pode filtrar por status, prioridade, buscar por termos específicos e realizar ações em massa.

---

## 📊 Regras de Negócio e Memória de Cálculo

### 1. Score de Urgência (Urgency Score)
- **O que é**: Uma nota de 0 a 100 gerada pela IA que avalia o quão crítico é o chamado com base no texto do cliente.
- **Como funciona**: A IA lê a descrição do problema e atribui a nota. Notas altas aparecem em vermelho para chamar atenção imediata.

### 2. SLA (Acordo de Nível de Serviço)
- **Resposta**: Tempo máximo para o primeiro retorno ao cliente.
- **Resolução**: Tempo máximo para resolver o problema definitivamente.
- **Status do SLA**:
    - 🔴 **Vencido**: O prazo expirou.
    - 🟡 **Atenção**: Está próximo de vencer.
    - 🟢 **No Prazo**: Dentro do limite combinado.

### 3. Busca Semântica
- **O que é**: Uma busca inteligente que entende o contexto, não apenas palavras-chave exatas.
- **Exemplo**: Se você buscar por "problema de login", ele pode encontrar chamados que dizem "não consigo acessar o sistema".

---

## 💡 Como Usar a Tela

1. **Fila Ativa**: Mostra a lista de chamados. Você pode clicar em qualquer linha para abrir a pré-visualização do chamado à direita.
2. **Ingestão Inteligente**: Aba para importar chamados em massa via planilha ou e-mail.
3. **Sincronizar E-mails**: Botão no topo para puxar novos chamados que chegaram por e-mail (via Power Automate).
4. **Ações em Massa**: Selecione vários chamados para alterar o status ou atribuir a um CSM de uma vez só.
